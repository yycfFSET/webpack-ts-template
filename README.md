## 介绍

你好，我是[燕云长风](https://yanyunchangfeng.github.io)。寓意：结合李白著名的边塞诗《关山月》取【燕云长风】—— 长风几万里，吹度玉门关。

## 用法

```
   pnpm i
   pnpm start
```

这是基于 webpack5,typescript 的配置模版 可快速搭建 ts 项目 配合[yycf-cli](https://github.com/yanyunchangfeng/yycf-cli) 使用更佳
